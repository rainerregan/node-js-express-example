# Node JS Example

NodeJS used for website backend service

## Prerequisites

To run this, you need these application installed:
* NodeJS

## Getting Started
1. Install NodeJs
2. Clone this repository
3. Cd into the sub directory
  ```sh
  cd node-js-express-example
  ```
4. Run `npm install`
5. Run `npm start`
5. Go to `http://localhost:3000`
